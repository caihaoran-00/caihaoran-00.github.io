奇门遁甲核心算法调研与实现

发表于 2025-08-29 分类于 code 阅读次数： 本文字数： 3.1k 阅读时长 ≈ 11 分钟

前言

背景：小程序核心算法

目的：记录+理思路+分享

正文

八字（四柱）

公历=阳历（现代通用），

八字（或称四柱），是中国传统命理学中的核心，顾名思义，就是八个字。这八个字分别代表了一个人出生时的年、月、日、时四个时间点的天干地支，每个时间点用一个天干和一个地支表示，合起来就是“四柱”（年柱、月柱、日柱、时柱），每柱两个字，共八个字。

核心概念：

  • 天干（10个）： 甲、乙、丙、丁、戊、己、庚、辛、壬、癸。
  • 地支（12个）： 子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥。
  • 组合规则： 天干和地支按顺序两两配对（阳干配阳支，阴干配阴支），从“甲子”开始，到“癸亥”结束，共有60种组合，称为“六十甲子”。

第一步：将公历日期转换为农历干支历日期

八字是基于中国的农历（干支历）计算的，而不是公历。所以第一步是转换。但现在我们不需要翻老黄历，可以通过已知的公式或规则来计算。

假设我们要计算：公历 1990年8月25日 上午10点 出生的人的八字。

第二步：计算年柱

年柱的确定以立春为界，而非公历的1月1日或农历的正月初一。
1990年的立春是2月4日。因为8月25日在立春之后，所以年柱确定为1990年的干支。

计算1990年的干支：
有一个简单的公式：

  1. 取公历年份数，减去3。
    1990 - 3 = 1987
  2. 将得到的差除以60（因为六十甲子循环），取余数。
    1987 ÷ 60 = 33 ... 余数为7
  3. 用这个余数去对应“六十甲子”序数（余数为0则是第60位“癸亥”）。
    余数7 对应的天干地支是第7组：庚午。
    • 天干算法：余数7 % 10 = 7。天干第7位是庚 (甲1、乙2、丙3、丁4、戊5、己6、庚7)。
    • 地支算法：余数7 % 12 = 7。地支第7位是午 (子1、丑2、寅3、卯4、辰5、巳6、午7)。

所以，年柱是：庚午。

第三步：计算月柱

二十四节气严格来说是由 “节” 和 “气” 交替组成的。在八字命理中，“节” 用于划分月份（月柱），而 “气” 则更多用于气候、农事和部分高级命理分析（如调候、命局寒暖燥湿的判断）。

二十四节气的公历日期不固定，但它的变化非常有规律，其背后依据的是纯粹的天文学，而非人为的历法规则。

以下是完整的二十四节气列表，并按“节”和“气”进行了分类。

月柱的确定同样以节气为分界，而不是农历的每月初一。

每个月由一個“节”开始，“节”是划分八字月柱的唯一标准，也称为“月令”或“节气”。一年从立春开始，共12个节：

  1. 立春 （寅月开始）
  2. 惊蛰 （卯月开始）
  3. 清明 （辰月开始）
  4. 立夏 （巳月开始）
  5. 芒种 （午月开始）
  6. 小暑 （未月开始）
  7. 立秋 （申月开始）
  8. 白露 （酉月开始）
  9. 寒露 （戌月开始）
  10. 立冬 （亥月开始）
  11. 大雪 （子月开始）
  12. 小寒 （丑月开始）

1990年8月25日，处在立秋（8月8日）和白露（9月8日）之间，所以是申月，这是地支，称为月支。

月柱的天干需要根据年柱的天干来推算，有一套固定的规则，称为“五虎遁”口诀：

“甲己之年丙作首，乙庚之岁戊为头。
丙辛必定寻庚起，丁壬壬位顺行流。
若问戊癸何方发，甲寅之上好追求。”

“五虎遁”里的 “虎”，指的就是地支 “寅”（因为寅虎）。所以“五虎遁”就是 “怎么找寅月天干” 的方法。

如果年份天干是… 口诀（密码） 那么正月（寅月）的天干就是… 正月就是…
甲 或 己 甲己之年 丙作首 丙 丙寅月
乙 或 庚 乙庚之岁 戊为头 戊 戊寅月
丙 或 辛 丙辛必定 庚起 庚 庚寅月
丁 或 壬 丁壬 壬位顺行流 壬 壬寅月
戊 或 癸 戊癸何方发，甲寅之上好追求 甲 甲寅月

我们的年干是庚（属于“乙庚”年），根据“乙庚之岁戊为头”，意思是乙年或庚年的正月（寅月）的月干是“戊”。

天干有10个：甲、乙、丙、丁、戊、己、庚、辛、壬、癸
地支有12个：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥

我们已经有了正月（寅月）：戊寅

现在，我们像数数一样，一个月一个月往下顺推（天干和地支都移到下一个）：

  1. 正月（寅月）：戊寅 （这是我们的起点）
  2. 二月（卯月）：天干“戊”的下一个是“己”，地支“寅”的下一个是“卯”。所以二月是 己卯。
  3. 三月（辰月）：天干“己”的下一个是“庚”，地支“卯”的下一个是“辰”。所以三月是 庚辰。
  4. 四月（巳月）：天干“庚”的下一个是“辛”，地支“辰”的下一个是“巳”。所以四月是 辛巳。
  5. 五月（午月）：天干“辛”的下一个是“壬”，地支“巳”的下一个是“午”。所以五月是 壬午。
  6. 六月（未月）：天干“壬”的下一个是“癸”，地支“午”的下一个是“未”。所以六月是 癸未。
  7. 七月（申月）：天干“癸”的下一个是“甲”（因为癸完了又从头循环到甲），地支“未”的下一个是“申”。所以七月（申月）就是 甲申。

推导过程可视化：

月支 寅 -> 卯 -> 辰 -> 巳 -> 午 -> 未 -> 申
月干 戊 -> 己 -> 庚 -> 辛 -> 壬 -> 癸 -> 甲
月柱 戊寅 己卯 庚辰 辛巳 壬午 癸未 甲申

现在到了月支的申，其对应的的甲就是我们要找的月干。

所以，月柱是：甲申。

第四步：计算日柱

这是计算中最复杂的一步，需要借助公式或直接查万年历。这里介绍一个适用于1900-2000年的公式。

计算公历1990年8月25日的日干支：

  1. 基础计算：

    • [(年尾二位数-1)*5 + (年尾二位数-1)/4 + 当年第几天 + 15] mod 60

      2001年 - 2100年的将+15去掉即可。

    • 代入：年尾二位数是 90

    • [89*5 + (90-1)//4 + 15 + 当年第几天] mod 60

    • [445 + 22 + 15 + 当年第几天] mod 60

    • [482 + 当年第几天] mod 60

  2. 计算当年第几天：

    • 1990年是平年（1990不能被4整除），2月有28天。
    • 1月(31) + 2月(28) + 3月(31) + 4月(30) + 5月(31) + 6月(30) + 7月(31) + 8月(25) = 31+28+31+30+31+30+31+25 = 237天

  3. 总和并取余：

    • 482 + 237 = 719
    • 719 mod 60 = 59

  4. 对应干支：

    • 余数59对应六十甲子第59位。天干：59 ÷ 10 = 5 ... 余数为 9 (0是癸，9是壬)。
    • 地支59 ÷ 12 = 4 ... 余数为 11(0是亥，11是戌)
    • 实际上，余数59对应的干支是“壬戌”。
    • 天干第9位是壬 (1甲,2乙,3丙,4丁,5戊,6己,7庚,8辛,9壬,10癸或0癸)。
    • 地支第11位是戌 (1子,2丑,3寅,4卯,5辰,6巳,7午,8未,9申,10酉,11戌,12亥或0亥)。

所以，日柱是：壬戌。

第五步：计算时柱

时柱由日柱的天干和出生的时辰共同决定。一天分为12个时辰，每个时辰2小时。

  • 上午10点属于 巳时 (上午9点至11点)。
时辰 地支 现代时间 别名 & 说明
夜半 子 (zǐ) 23:00 - 01:00 子夜、三更天，是一天的开始
鸡鸣 丑 (chǒu) 01:00 - 03:00 四更天
平旦 寅 (yín) 03:00 - 05:00 黎明、五更天
日出 卯 (mǎo) 05:00 - 07:00 又名“日始”、“破晓”
食时 辰 (chén) 07:00 - 09:00 古人吃早饭的时间
隅中 巳 (sì) 09:00 - 11:00 临近中午
日中 午 (wǔ) 11:00 - 13:00 正午、中午
日昳 未 (wèi) 13:00 - 15:00 又名“日跌”，午后

黄昏 戌 (xū) 19:00 - 21:00 一更天
人定 亥 (hài) 21:00 - 23:00 二更天，该安歇睡觉了

时柱的地支固定为“巳”。时柱的天干需要根据日柱的天干来推算，有一套“五鼠遁”口诀：

“甲己还加甲，乙庚丙作初。
丙辛从戊起，丁壬庚子居。
戊癸何方发，壬子是真途。”

我们的日干是壬（属于“丁壬”日），根据“丁壬庚子居”，意思是丁日或壬日的子时（23:00-1:00）的时干是“庚”。
所以：

  • 子时 (23-1): 庚子
  • 丑时 (1-3): 辛丑
  • 寅时 (3-5): 壬寅
  • 卯时 (5-7): 癸卯
  • 辰时 (7-9): 甲辰
  • 巳时 (9-11): 乙巳

所以，时柱是：乙巳。

最终结果

将以上四柱汇总，这个出生于公历1990年8月25日上午10点的人的八字（四柱）是：

  • 年柱：庚午
  • 月柱：甲申
  • 日柱：壬戌
  • 时柱：乙巳

所以他的八字就是：庚午 甲申 壬戌 乙巳

算一算

咱们来手动算算2025-08-20 17:43的八字：

  • 首先肯定过了立春了，按2025年算
  • 2025-3=2022,2022÷60=33 … 余42，42 % 10 = 2 –>天干：乙，42 % 12 = 6—>地支：巳（已巳）
  • 8-20是立秋与白露之间–>地支：申，五虎遁–>天干：甲（甲申）
  • [（25-1）*5 +（25-1）/4] +31+28+31+30+31+30+31+20 = 358，358 mod 60 = 58，58 mod 10 = 8, 58 mod 12 = 10，辛酉
  • 时柱的地支为酉，日干是辛–>丁酉

所以：

  • 年柱：已巳
  • 月柱：甲申
  • 日柱：辛酉
  • 时柱：丁酉

总结与注意事项：

  1. 节气的重要性：年和月的分界严格依据立春和节气，绝对不能按公历或农历的月初年底简单划分。
  2. 日柱计算的复杂性：日柱的公式计算较为繁琐，且1900年之前和2000年之后的公式有修正。
  3. 时区问题：八字采用的都是北京时间（东八区时间）。如果出生在其他时区，需要进行换算。例如，新疆出生的人可能需要将当地时间换算为北京时间后再排盘（命理学界对此有不同看法，但传统以北京时间为准）。
  4. 夏令时：如果出生在实行夏令时的时期（1986-1991年），需要将表上的时间减去1小时后再计算时辰。1990年有夏令时，上午10点实际上是标准时间上午9点，那么时辰就应该是辰时（7-9点），时柱就变成了甲辰。这一点非常重要且常被忽略！
