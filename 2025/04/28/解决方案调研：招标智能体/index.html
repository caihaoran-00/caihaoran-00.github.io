解决方案调研：招标智能体

发表于 2025-04-28 更新于 2025-04-30 分类于 theory 阅读次数： 本文字数： 2.8k 阅读时长 ≈ 10 分钟

需求

  1. 网页自动爬虫智能体：从各大公共交易平台提取项目招标网页数据和下载各类招标信息附件；

  2. 数据提取智能体模型：下载下来的各类招标信息附件(PDF、word、excel、图片等)进行文件解析并按关键词把非结构化数据转换成结构化数据，经清洗校验后存入公司数据库，如：项目编号、项目名称、工程概况、招标概况、开标日期、开标地点、招标控制价等；

  3. 投标报价预测智能体模型:利用历史招标数据对类拟项目进行模拟推算，预测投标报价中标概率；

需求分析与大致方案

这是个非常典型但也很有挑战的 三阶段智能体系统，分别是 数据抓取 → 数据加工 → 数据建模与推理。

总体架构

三个智能体模块将以模块化方式设计，彼此独立但通过数据流和接口协作。整体流程如下：

  1. 网页自动爬虫智能体：从公共交易平台抓取招标网页数据和附件，存储原始数据；
  2. 数据提取智能体：解析附件，提取关键字段，清洗并结构化后存入数据库；
  3. 投标报价预测智能体：基于历史数据和提取的结构化数据，训练预测模型，输出中标概率。

背景知识

以客户提供的《浙江省房屋建筑和市政基础设施施工招标文件示范文本（适用于杭州市资格后审项目）（2023版）》以及《半山单元高中（暂名）施工项目招标文件》的相关页面内容为例，梳理一下投标、开标、评标的流程：

一、 投标流程 (Bidding/Tendering Process - 投标人视角)

这是潜在投标人参与项目竞争的过程，主要步骤如下：

  1. 获取招标文件：
    • 潜在投标人需要在规定的时间内（本项目未明确截止获取时间，但通常在招标公告中说明），通过指定的平台，即杭州建设工程招标造价平台 (https://ztb.cxjw.hangzhou.gov.cn:8092) 和 杭州市公共资源交易平台 (http://hzctc.hangzhou.gov.cn) 获取招标文件及相关资料（如图纸、清单、招标控制价明细等）。（Page 4, 2.2.1）
  2. 研究招标文件并提疑：
    • 投标人仔细阅读招标文件，理解项目要求、资格条件、技术规范、合同条款、评标办法等。
    • 如有疑问，必须在2025年02月23日前，通过杭州建设工程招标造价平台以实名形式在线提交澄清要求（提疑）。逾期提交，招标人可拒绝受理。（Page 4）
  3. 获取招标文件的澄清与修改：
    • 招标人会在2025年02月26日17:00前，通过上述两个平台发布招标文件的澄清、答疑或修改文件（招标补充文件）。
    • 投标人有责任自行关注并下载这些文件，招标人不再单独通知。因投标人自身贻误导致投标失败，责任自负。（Page 4, 2.2.1）
  4. 编制投标文件：
    • 投标人根据招标文件及其所有补充文件的要求，使用指定的电子投标文件制作工具（V3.2.0版本）编制电子投标文件，生成后缀名为“.HzTbs”的文件。（Page 7, 3.7.3(2)）
    • 投标文件主要包含四个部分：资格审查材料、资信标、技术标、商务标。（Page 7, 3.7.3(2)）
    • 特别注意： 本项目的技术标采用暗标评审，投标人在制作技术标暗标文件时，不得出现任何能直接或间接识别投标人身份的信息（如公司名称、人员姓名、特定logo等）。（Page 5, 3.1.2; Page 7, 3.7.3(2) point 3）技术标文件宜控制在300页以内。（Page 5, 3.1.2）
    • 需要按要求填写业绩汇总表等特定格式文件。（Page 7, 3.7.3(3)）
    • 按照要求进行电子签章（CA数字证书加盖单位和法定代表人电子印章）。（Page 6, 3.7.3(1)）
  5. 缴纳投标保证金：
    • 在投标截止时间前，按要求足额缴纳人民币50万元的投标保证金。（Page 5, 3.4.1）
    • 缴纳方式可以是银行保函、保证保险、担保公司担保、转账或数字保函。采用转账方式的，必须从投标人基本账户转出，并通过杭州银行“杭银在线”系统与本项目关联确认。（Page 5-6）
  6. 递交投标文件：
    • 在**投标截止时间（2025年03月06日 14:00:00）*前，使用专用密钥（CA锁）将加密的电子投标文件（.HzTbs文件）上传至*杭州市公共资源交易平台。（Page 7, 4.1.1, 4.2.1, 4.2.2）
    • 逾期上传、未按要求加密或无法解密的投标文件将被拒收。（Page 8, 4.2.5）

二、 开标流程 (Bid Opening Process)

开标是在投标截止时间的同一时间公开进行的过程，主要步骤如下：

  1. 时间和地点：
    • 开标时间：2025年03月06日 14:00:00。（Page 7, 4.2.1）
    • 开标地点：杭州市公共资源交易中心第8开标室（临江金座2号楼）。（Page 8, 5.1）
    • 开标平台：杭州市公共资源交易平台。（Page 8, 5.1）
  2. 解密：
    • 开标时，招标人代表使用招标CA数字证书对投标人上传的加密电子投标文件进行解密。（Page 8, 5.2(一)）
    • 投标人也需要在此环节使用自己的CA锁远程解密，或按要求到现场解密（具体操作看平台指引）。
  3. 信息公开（唱标）：
    • 投标文件成功解密并导入评标系统后，投标单位名称、项目负责人、投标报价、工期等主要信息将在电子交易平台公开显示。（Page 8, 5.2(二)）
    • 系统会采用投标函信息作为签到信息，标录显示问题仅作提醒，不影响资格审查或否决投标。（Page 8, 5.2(二)）
  4. 现场随机抽取（确定评标参数）：
    • 在开标会现场，招标人会随机抽取评标过程中需要用到的一些参数，包括但不限于：（Page 8-9）
      • 综合单价评审项目（10项）
      • 综合单价评审合理区间K值（本项目为15%）
      • 价格入围方式（本项目采用技术标打分法，开标时可能抽取价格分区方式及高低区间系数KH, KL）
      • 技术标打分法下的最佳报价下浮系数（在1%-3%之间抽取，5个选项）

注意：开标环节不进行评分，只公开基本信息和确定后续评标所需的部分参数。

三、 评标流程 (Bid Evaluation Process)

开标后，由依法组建的评标委员会对投标文件进行评审，最终确定中标候选人。主要步骤如下：

  1. 组建评标委员会：
    • 由招标人依法组建，成员为5人及以上单数，包括技术、经济等方面的专家，招标人代表按规定比例参与，专家需随机抽取。（Page 9, 6.1.1）
  2. 确定评审区间（若投标人>15家）：
    • 按Page 22-24所述的信用择优、招标人推荐、价格入围相结合的方式，确定15家投标人进入详细评审。
  3. 资格审查：
    • 评标委员会首先对进入评审区间的投标人进行资格审查，核对是否满足招标文件（包括招标公告和投标人须知前附表1.4.1、10.1等）规定的所有资格条件。（Page 11-12, Page 24）
    • 审查内容非常细致，包括资质、人员、业绩、信誉、投标保证金、动态核查、有无违法限制投标等情况。任何一项不满足都可能导致否决投标。（Page 11-12）
    • 若因资格审查导致有效投标人不足15家，会按规则进行递补。（Page 24）
  4. 详细评审（技术标打分制综合评估法）：
    • 计算最佳报价： 对通过资格审查的有效投标人的报价进行算术平均，再下浮一个开标时随机抽取的百分比（1%-3%之间），得到最佳报价。（Page 24）
    • 技术标评审（暗标，30分）： 专家对技术标的暗标部分进行打分，评审因素包括总体部署、主要施工方案、质量/安全/工期/环保措施、重点难点分析、新技术应用等。（Page 25）可能还包括对项目负责人的陈述和答辩环节进行评分（需招标文件明确，Page 26）。
    • 资信标评审（10分）： 根据企业类似业绩、项目负责人类似业绩、企业信用等级、项目负责人和班子成员能力（如职称、证书、年龄）等进行评分。（Page 26-29）
    • 商务标评审（60分）：
      • 投标报价得分（56分）：根据投标报价与最佳报价的偏离程度，按规定公式计算得分。（Page 29）
      • 工程量清单符合性评审（4分）：对随机抽取的综合单价进行评审，看其是否落在合理区间（基准价的±K%）内，超出区间的扣分。（Page 30）
    • 初步评审（符合性检查）： 除了资格审查，还包括对投标文件的格式、签字盖章、投标函填写、是否响应实质性要求（如工期、质量标准）等的检查。（Page 13）
  5. 汇总得分与排名：
    • 将技术标、资信标、商务标得分相加，得到总得分。
    • 按总得分由高到低进行排序。若总分相同，依次按投标报价低者优先、资信标得分高者优先、技术标得分高者优先的原则排序；若仍相同，则抽签决定。（Page 30）
  6. 推荐中标候选人：
    • 评标委员会根据排序结果，推荐排名第一的投标人为中标候选人（本项目推荐1名）。（Page 9, 6.3.2）
  7. 定标前核查：
    • 招标人在确定中标人前，会对中标候选人及其项目负责人进行核查，包括安全生产许可证、资质动态核查、失信记录、行贿犯罪记录等。（Page 15）
  8. 确定中标人：
    • 核查通过后，招标人依法确定中标候选人为中标人。（Page 9, 7.2.1）
  9. 公示与通知：
    • 中标候选人（会公示得分情况）和中标结果会在指定平台进行公示。（Page 9-10）
