相信有小友也看到过openai.ChatCompletion.create()这个方法，但有时候看到的又是client.chat.completions.create()方法，那么这两种方法有什么区别和联系呢？

对比项 openai.ChatCompletion.create() client.chat.completions.create()
模型支持 只能调用 OpenAI 的模型（gpt-4, gpt-3.5） 可以调用任何兼容 OpenAI API 格式的模型（如通义 Qwen、Mistral、Moonshot、DeepSeek）
base_url 设置方式 不支持设置 base_url，只能发给 api.openai.com 可以显式设置 base_url 指向其他服务
可扩展性 低 高（可以用来调用 OpenRouter、AnyScale、DashScope 等）
推荐用途 调用 OpenAI 官方服务 调用任何支持 OpenAI API 格式的第三方模型（如阿里通义 Qwen）
